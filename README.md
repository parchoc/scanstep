# Приложение для расчёта параметров стопы

Данное приложение позволяет разменить стопу по изображениею и расчитать её параметры по методу А.Н. Герасевича. Она позволяет найти параметры:
* длина стопы;
* ширина стопы;
* ширина пятки;
* ∠ α;
* ∠ β;
* ∠ γ;
* угол Кларка;
* коэффициент Чижина;
* коэффициент Вайсфлога.
